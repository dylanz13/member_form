About This Project

This project is a simple member list application demonstrating basic (pure) HTML, CSS, and JavaScript skills. It includes:

  • A form to add new members with fields for name, description, age, image URL, and favorite hobby. Name, description, and image URL are required fields.
  • Visual indication (red asterisk) for required fields and a red outline for empty required fields after they lose focus.
  • A list that dynamically updates with new member cards containing the entered information.
  • Individual delete buttons for each member card and an animation for adding and removing cards.
  • A delete all button to clear the member list and a search functionality to filter members.
